### 리스트 컴프리헨션(List Comprehension)
- 반복문을 한줄로 



```python
fruits = ['apple','kiwi','mango']
newlists = []
# 알파벳 a가 있는 과일만 추출 후, 새로운 리스트에 담기
for fruits in fruits:
  #print(fruits)
  if "a" in fruits:
    newlists.append(fruits)
print(newlists) 
```

    ['apple', 'mango']
    


```python
# 리스트 컴프리헨션
fruits = ['apple','kiwi','mango']
newlists = []
newlist = [fruits for fruits in fruits if 'a' in fruits]
print(newlist)
```

    ['apple', 'mango']
